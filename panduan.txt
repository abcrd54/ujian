Buatkan panduan program untuk sistem ujian sekolah digital.

Tech stack:
- Frontend: JavaScript framework modern, gunakan Next.js atau React + Vite
- Backend/API: Next.js API Routes / Express.js
- Database: Supabase PostgreSQL
- Auth: Supabase Auth
- Storage: Supabase Storage
- UI: Tailwind CSS + shadcn/ui
- Deployment: Vercel untuk frontend, Supabase untuk database

Konsep aplikasi:
Sistem ujian sekolah berbasis web/app dengan dashboard admin modern minimalis.
Dashboard digunakan admin sekolah untuk mengelola guru, siswa, kelas, mapel, ujian, jadwal ujian, tugas pembuatan soal, review soal, hasil ujian, dan integrasi API.

Role user:
1. Super Admin
2. Admin Sekolah
3. Guru
4. Siswa

Hak akses:
Super Admin:
- Kelola data sekolah
- Monitoring semua sekolah
- Kelola API key sekolah

Admin Sekolah:
- Kelola guru
- Kelola siswa
- Kelola kelas
- Kelola mapel
- Membuat ujian

Guru:

- Membuat soal pilihan ganda dan essay

Siswa:

- Melihat jadwal ujian

Flow program:
1. Super Admin membuat sekolah.
2. Sistem generate API key untuk sekolah.
3. Admin sekolah login.
4. Admin input guru, siswa, kelas, mapel.
5. Admin membuat ujian.
6. Admin memilih guru pembuat soal.
7. Guru login dan melihat tugas ujian.
8. Guru membuat soal pilihan ganda dan essay.
9. Guru submit soal ke admin.
10. Admin review dan approve soal.
11. Admin membuat jadwal ujian.
12. Jadwal publish ke aplikasi siswa.
13. Siswa login dan mengerjakan ujian.
14. Jawaban pilihan ganda dinilai otomatis.
15. Jawaban essay dikoreksi guru.
16. Nilai final muncul di hasil ujian.

API key:
- API key dibuat per sekolah
- Digunakan untuk integrasi aplikasi mobile/web siswa
- API key disimpan di tabel schools
- Request wajib memakai header:
  Authorization: Bearer {API_KEY}
  X-School-ID: {SCHOOL_ID}

Keamanan:
- Gunakan Supabase RLS
- User hanya bisa akses data berdasarkan school_id
- Guru hanya bisa akses ujian yang ditugaskan
- Siswa hanya bisa akses sesi ujian miliknya
- API key tidak ditampilkan penuh setelah dibuat
- Gunakan rate limit pada endpoint publik
- Validasi waktu ujian sebelum siswa bisa mulai
- Auto submit jika waktu habis
